# 2D Action Framework - Core System

##  Overview
量産性とメンテナンス性を重視した2Dゲーム開発基盤です。

##  Technical Features
- **シーン管理**: シングルトンパターンによる永続的なマネージャー実装
- **UIシステム**: Overridesを活用したリザルト画面のテンプレート化
- **AI-Driven Development**: Gemini 3 Flashを開発パートナーに採用し、設計の壁打ちを実施
